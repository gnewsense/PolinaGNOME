# PolinaGNOME 🐧

Welcome to **PolinaGNOME**, a pure, optimized, GNOME-only Linux desktop experience built entirely on top of the **Debian Universal Operating System**.

## 🛑 Important Notice & Disclaimer
PolinaGNOME is an independent project. It is **100% Debian-based** and has **no technical connection to Ubuntu**. 

The **uG logo** utilized in this project is an intentional visual tribute to the historic, discontinued Ubuntu GNOME project. In PolinaGNOME:
* **u** stands for **Universal** (honoring Debian's official title: *The Universal Operating System*).
* **G** stands for **GNOME**.

This project is **NOT** affiliated with, sponsored by, or endorsed by Canonical Ltd. or the original Ubuntu GNOME team. 

## Features
* Built on stable, secure Debian packages.
* Pure, bloat-free Vanilla GNOME desktop environment.
* Nostalgic, clean blue user interface styling.
